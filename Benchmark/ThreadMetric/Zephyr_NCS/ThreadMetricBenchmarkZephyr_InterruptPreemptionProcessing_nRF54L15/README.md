# ThreadMetricBenchmarkZephyr_InterruptPreemptionProcessing_nRF54L15

Standalone NCS/Zephyr wrapper for the official Zephyr Thread-Metric scenario `benchmark.thread_metric.interrupt_preemption` on `nrf54l15dk/nrf54l15/cpuapp`.

Build from an NCS workspace root or anywhere with `west` and `ZEPHYR_BASE` available:

```bash
west build -p always -b nrf54l15dk/nrf54l15/cpuapp ThreadMetricBenchmarkZephyr_InterruptPreemptionProcessing_nRF54L15
west flash
```
